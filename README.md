
# 🪔 Login Lamp UI

An interactive login page with a rope-pull lamp toggle.

## Features
- Drag the rope to toggle the lamp on/off
- Warm golden glow effect when lamp is on
- Spring-back rope animation
- Web Audio API click sound
- Touch & mouse support

## Tech Stack
HTML · CSS3 · Vanilla JavaScript · Web Audio API
